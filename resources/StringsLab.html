String Lab

Directions

Practice solving each of the problems below, using the values provided as input to return the expected outputs.

Starter Code

Java JavaScript Python

Practice Problems

  1. Counting Letters

    Description: Given a string word, write a function that returns the number of letters in word.

    Examples:

    No Input Output
    1 Hello 5
    1 function 8
    1 Supercalifragilisticexpialidocious 34

  2. Counting Twice

    Description: Given two strings a and b, write a function that returns the total number of letters in both strings.

    Examples:

    No Input Output
    1 hello, world 10
    1 string, function 14
    1 Howdy, Hola 9

  3. Alphabetically Order

    Description: Given two strings a and b, write function that returns the string that comes first alphabetically, ignoring capitialization.

    Examples:

    No Input Output
    1 hello, world hello
    1 Reverse, Words Reverse
    1 Woodchucks, chucking chucking

  4. Counting Vowels

    Description: Given a string word, write a function that returns the number of vowels in word. Always treat y as a consonant.

    Examples:

    No Input Output
    1 Hello 5
    1 function 3
    1 Supercalifragilisticexpialidocious 16

  5. Counting Consonants

    Description: Given a string word, write a function that returns the number of consonants in word. Always treat y as a consonant.

    Examples:

    No Input Output
    1 Hello 3
    1 function 5
    1 Supercalifragilisticexpialidocious 18

  6. All Uppercase

    Description: Given a string word, write a function that returns word with every letter capitalized.

    Examples:

    No Input Output
    1 Hello HELLO
    2 World WORLD
    3 ALLCAPS ALLCAPS

  7. All Lowercase

    Description: Given a string word, write a function that returns word with every letter lowercase.

    Examples:

    No Input Output
    1 Hello hello
    2 WORLD world
    3 lowercase lowercase

  8. Case Swap

    Description: Given a string word, write a function that returns word with the case of each letter switched.

    Examples:

    No Input Output
    1 Hello hELLO
    2 WORLD world
    3 AbCdEf aBcDeF

  9. Caesar Cypher

    Description: Given a string word and an int shift, write a function that returns word with each letter replaced by the letter shift positions to the right i the alphabet.

    Examples:

    No Input Output
    1 Hello, 3 Khoor
    2 World, 26 World
    3 Buzz, 1 Cvaa

  10. Counting Words

    Description: Given a string sentance, write a function that returns the number of words in the sentance.

    Examples:

    No Input Output
    1 What is the weather forecast today? 6
    2 Write a function that returns the number of words in the sentance. 12
    3 How much wood would a woodchuck chuck if a woodchuck could chuck wood? 13

  11. Capital Words

    Description: Given a string sentance, write a function that returns sentance with the first letter of every word capitalized.

    Examples:

    No Input Output
    1 hello world Hello World
    2 capitalizing strings of data Capitalizing Strings Of Data
    3 Howdy Y'all! Howdy Y'all!

  12. Subsection

    Description: Given a string sentence and two ints a and b, write a function that returns all the letters in sentence between the postions of a and b.

    Examples:

    No Input Output
    1 hello world, 3, 8 lo wo
    2 sentence, 0, 4 sent
    3 Howdy Y'all!, 6, 7 Y

  13. String Reverse

    Description:Given a string s, write a function that returns the value of s reversed.

    Examples:

    No Input Output
    1 Hello olleH
    2 Reversing strings of data atad fo sgnirts gnisreveR
    3 How much wood would a woodchuck chuck if a woodchuck could chuck wood? ?doow kcuhc dluoc kcuhcdoow a fi kcuhc kcuhcdoow a dluow doow hcum woH

  14. Word Reverse

    Description: Given a string sentance, write function that reverses each word in sentance.

    Examples:

    No Input Output
    1 Hello world olleH dlrow
    2 Reverse Words esreveR sdroW
    3 Woodchucks chucking wood skcuhcdooW gnikcuhc doow

  15. Sentence Reverse

    Description: Given a string sentance, write function that reverses the order of the words in sentance.

    Examples:

    No Input Output
    1 Hello world world Hello
    2 Reverse Words Words Reverse
    3 Woodchucks chucking wood wood chucking Woodchucks
