Maps Lab

Directions

Practice solving each of the problems below, using the values provided as input to return the expected outputs.

Starter Code

Java JavaScript Python

Practice Problems

  1. Number of Keys in a Map

    Description: Given a Map called myMap, write a function that returns the number of keys in myMap.

    Examples:

    No Input Output
    1 {alpha: 1, beta: 2, gamma: 3} 3
    2 {} 0
    3 {alpha: 1, beta: 2, gamma: 3, delta: 4, epsilon: 5} 5

  2. Remove Key from Map

    Description: Given a Map called myMap and a key, write a function that removes the key from myMap if it exists and returns myMap.

    Examples:

    No Input Output
    1 {alpha: 1, beta: 2, gamma: 3}, alpha {beta: 2, gamma: 3}
    2 {alpha: 1, beta: 2, gamma: 3}, gamma {alpha: 1, beta: 2}
    3 {alpha: 1, beta: 2, gamma: 3}, delta {alpha: 1, beta: 2, gamma: 3}

  3. Create Map of Zeros

    Description: Given an array called names, create a map where each name in names is a key in the map. The value for each key should be 0.

    Examples:

    No Input Output
    1 [Kaleb, Jess, Riley] {Kaleb: 0, Jess: 0, Riley: 0}
    2 [alpha, beta, gamma] {alpha: 0, beta: 0, gamma: 0}
    3 [] {}

  4. First and Second Letters as Key-Value Pairs

    Description: Given an array called words, create a map where the key is the first letter of each word and the value is the second letter. If two words start with the same letter, use the first word to determine the value.

    Examples:

    No Input Output
    1 [You, are, smart] {y:o, a:r, s:m}
    2 [alpha, beta, gamma] {a:l, b:e, g:a}
    3 [Back, to, Boston] {b:a, t:o}

  5. Letter Occurrence in a Sentence

    Description: Given a String called sentence, determine how many times each letter occurs in the sentence. Return your answer as a Map of letters as keys and the occurrence of that letter as the value.

    Examples:

    No Input Output
    1 You are smart {a:2, e:1, m:1, o:1, r:2, s:1, t:1, u:1, y:1}
    2 Alpha beta gamma {a:5, b:1, e:1, g:1, h:1, l:1, m:2, p:1, t:1}
    3 Nada para ti! {a:4, d:1, i:1, n:1, p:1, r:1, t:1}

  6. Unique Words in a Sentence

    Description: Given a String called sentence, return a Map where keys are each word in the sentence and the value is true if the word occurs only once in the sentence, false otherwise.

    Examples:

    No Input Output
    1 You are smart {you: true, are: true, smart: true}
    2 Which witch is which? {which: false, witch: true, is: true}
    3 Nada para ti! {nada: true, para: true, ti: true}

  7. Remove Duplicate Values in a Map

    Description: Given a Map called myMap, if any keys share the same value, keep the key that comes alphabetically first and remove all the other keys. Return the updated Map.

    Examples:

    No Input Output
    1 {alpha: 1, beta: 2, gamma: 1} {alpha: 1, beta: 2}
    2 {alpha: 1, beta: 2, gamma: 3} {alpha: 1, beta: 2, gamma: 3}
    3 {alpha: 1, beta: 1, gamma: 1} {alpha: 1}

  8. Remove Duplicates from Array

    Description: Given an array called myArray, remove all duplicate elements in myArray and return the result.

    Examples:

    No Input Output
    1 [You, are, smart] [You, are, smart]
    2 [alpha, beta, alpha] [alpha, beta]
    3 [alpha, alpha, alpha] [alpha]

  9. Common Letters in Two Strings

    Description: Given two Strings called alpha and beta, return a new String that contains letters that only appear in both alpha and beta. The letters in the output should be in alphabetical order.

    Examples:

    No Input Output
    1 alpha, beta a
    2 alphabet, alphabet abehlpt
    3 dog, cat *Empty String*

  10. Check If Goal Can Be Formed from Provided Letters

    Description: Given two Strings called provided and goal, return true if you can create goal using the letters provided in provided.

    Examples:

    No Input Output
    1 alpha, beta false
    2 state, taste true
    3 you are smart, my rat true
