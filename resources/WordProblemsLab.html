Directions

Practice solving each of the problems below, using the values provided as input to return the expected outputs.

Practice Problems

  1. Arrange Students by Height

    Description: A teacher wants to arrange her students by height. Provided an array, where each element represents the height of a student, return the array ordered from shortest to tallest student.

    Examples:

    No Input Output
    1 [4.25, 3.5, 5.5, 5] [3.5, 4.25, 5, 5.5]

  2. Determine Most Common Letter Grade

    Description: The professor wants to know the most common letter grade on the midterm exam. Given the grade points of each student and using a 10-point scale, calculate the most common letter grade received.

    Examples:

    No Input Output
    1 {Jorge: 89, Riley: 72, Valerie: 84, Hermione: 100} B

  3. Dog Walker

    Description: You started a dog walking business. Given an array containing all the dogs represented by their size, determine how many trips it will take to walk all the dogs. If two dogs are the same size, they can be walked together; otherwise, they must be walked separately.

    Examples:

    No Input Output
    1 [3,2,2,3,6] 3
    1 [1,5,4,3,2] 5

  4. Determine Distance from Pole

    Description: A drunken man stumbles randomly. His steps are represented by 'F' for forward and 'B' for backward. Each step moves him exactly one foot in that direction. Given an array representing all the steps the man took, determine how far he is from the pole.

    Examples:

    No Input Output
    1 [F, F, F, B, F, F] 4
    1 [B,B] 2

  5. Pair Students by Age

    Description: You are tasked with pairing up all the students in preparation for the field trip. You are told that the students should be paired up by age. Given all the ages of the students, determine how many do not have a partner the same age to be matched with.

    Examples:

    No Input Output
    1 [5,5,6,5,6] 1
    1 [8,8,7,7] 0

  6. Meerkat Distance from Burrow

    Description: A Meerkat peeks out of her burrow to look for food. She can move north, south, east, or west, represented by 'N', 'S', 'E', 'W', and any movement is comprised of these directions. Determine how far the Meerkat is from her home after all her moves.

    Examples:

    No Input Output Explanation
    1 [N4, W1, S1, E6] 5 N3 and E5 make a 5ft diagonal in the NE direction
    1 [N1, N1] 2

  7. Walk Across Stone Steps

    Description: Given the number of stone steps in the pond, determine how many possible ways you could walk across the pond. You could step on every stone, step over every other stone, or mix and match.

    Examples

    No Input Output Explanation
    1 3 3 * single step, single step, single step
    * skip Step, single step
    * single step, skip step
    1 4 5 * single step, single step, single step, single step
    * skip Step, single step, single step
    * single step, skip step, single step
    * single step, single step, skip step
    * skip step, skip step

  8. Decode Secret Message

    Description: You are provided a top secret message consisting of only the numbers 0-9. The numbers in the message map to letters in the following way: A is 1, B is 2, .... and Z is 26. Determine how many possible ways the message can be decoded.

    Examples:

    No Input Output Explanation
    1 127 2 ABG, LG
    1 1234 3 ABCD, LCD, AWD
