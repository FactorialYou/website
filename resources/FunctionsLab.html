Functions Lab

Directions

Practice solving each of the problems below. Additionally, solve the problems without using console input (Scanner, input, cin, etc) or output (System.out.print, print, cout, etc) as part of your function. Instead write functions that accept inputs as parameters and provides outputs as return values.

                  
// Do this
public int mySolution(int param){
  return param * 2;
}

int answer = mySolution(3);
System.out.println(answer);
                  
                
                  
 // Not this
public void mySolution(int param){
  System.out.println(param * 2);
}

// Or this
public void mySolution(){
  Scanner scanner = new Scanner(System.in);
  int num = scanner.nextInt();
  System.out.println(num * 2);
}

    Eyes: BLUE

    No Input Output

  20. Fibonacci

    Examples:

    No Input Output
    1 3 True
    2 2 True
    3 4 False
