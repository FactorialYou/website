Strings Utils Lab

Directions

List below are a number of methods that come bundled with the String class. The goal here is to implement them from scratch. You are allowed to use only three built-in methods, charAt(), length(), and concat().

For example, if I was given the method String getFirst(String str) and told it should return the first character of str or null if str is empty, i would write the following function:

                    
String getFirst(String str){
    if(str.length() !== 0){
        return str.get(0);
    }
    else{
        return null;
    }
}

Methods

No Method Signature Description
1 int compareTo(String strA, String StrB) Compares two strings alphabetically. If strA come first alphabetically, return a negative number. If strB comes first alphabetically, return a positive number. If strA and strB are the same, return 0.
2 boolean contains(String original, String subString) Returns true if and only if orignal contains subString.
3 boolean endsWith(String str, String suffix) Tests if str ends with the specified suffix.
4 boolean equals(String strA, String strB) Compares strA to strB returning true if they are the same string and false otherwise.
5 int indexOf(String str, int ch) Returns the index within str of the first occurrence of the specified character
6 boolean isEmpty(String str) Returns true if, and only if, str has a length of 0
7 String repeat(String str, int count) Returns a string whose value is the concatenation of str repeated count times
8 String replace(String str, char oldChar, char newChar) Returns a string resulting from replacing all occurrences of oldChar in str with newChar
9 String[] split(String str, String subString) Splits str around matches of subString, returning an array of the split contents of str
10 boolean startsWith(String str, String prefix) Tests if str starts with the specified prefix
11 String strip(String str) Returns a string whose value is str, with all leading and trailing white space removed
12 String substring(String str, int beginIndex) Returns a string that is a substring of str, starting at beginIndex and stoping at the end of str
13 String substring(String str, int beginIndex, int endIndex) eturns a string that is a substring of str, starting at beginIndex and stoping at, but not including, endIndex.
14 String toLowerCase(String str) Converts all of the characters in str to lower case.
15 String toUpperCase(String str) Converts all of the characters in str to upper case.
